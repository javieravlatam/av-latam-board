# reports/

Carpeta destinada a informes ejecutivos (PDF) generados por el prototipo SIC-AV.

El prototipo no guarda archivos aquí automáticamente: el Informe Ejecutivo de Gestión Comercial se genera con `window.print()` desde el navegador (botón "Descargar informe (PDF)" en `sic_chile.html` / `sic_peru.html`), y es el propio usuario quien decide dónde guardarlo al usar la opción "Guardar como PDF" del diálogo de impresión. Esta carpeta existe como ubicación sugerida para esos informes durante pruebas y demostraciones internas.

No subir aquí informes con datos reales de vendedores mientras el sistema siga en fase de prototipo — ver advertencias en `../README.md`.
